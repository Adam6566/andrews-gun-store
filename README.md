## ✨ Features

### 🗂️ Complete Firearms Catalog
- **150+ firearms** from every era and nation
- Organized by category: Pistols, Revolvers, Rifles, Assault Rifles, SMGs, Shotguns, Machine Guns, Snipers, Historical
- Search by name, caliber, country, or era
- Filter by weapon type with live counts

### 📖 AI-Powered Gun Details
Click any firearm to access:
- **History Tab** — Full origin story, designers, military service, famous conflicts, legacy
- **Parts Breakdown** — Every component explained (barrel, bolt, trigger, magazine, etc.)
- **Variants & Service** — All models, countries that used it, current status

### ⚙️ Custom Firearm Builder
Design your own weapon:
- Choose base type, caliber, barrel length, frame material, finish, and special features
- AI generates a complete spec sheet and description
- Test-fire your creation

### 📚 Firearms Library
- Ask the AI any question about firearms history or mechanics
- Quick-access curated questions
- Comprehensive encyclopedia-style answers

### 🔫 Realistic Gunshot Sounds
- Web Audio API synthesized gunshot for every weapon category
- Unique sound signatures: pistols, revolvers, rifles, shotguns, machine guns, snipers
- Muzzle flash visual effect
- Press SPACEBAR when viewing a gun detail to fire!

## 🚀 Deployment on GitHub Pages

### Step 1: Create GitHub Repository
1. Go to [github.com](https://github.com) and sign in
2. Click **New Repository**
3. Name it: `andrews-gun-store`
4. Set to **Public**
5. Click **Create Repository**

### Step 2: Upload Files
1. Click **uploading an existing file**
2. Drag `index.html` into the upload area
3. Click **Commit changes**

### Step 3: Enable GitHub Pages
1. Go to **Settings** → **Pages**
2. Under **Source**, select `Deploy from a branch`
3. Choose `main` branch, `/ (root)` folder
4. Click **Save**

### Step 4: Access Your App
Your app will be live at:
```
https://YOUR-USERNAME.github.io/andrews-gun-store/
```
